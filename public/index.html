Comix

Base Import
  1. Download the MangaUpdates Links file (.txt) above.

  4. Upload the .txt file.

MyAnimeList

Via Comix
  1. Pre-requisite: Import to Comix first (see left).
  2. Go to Comix Export .
  3. Select MAL XML and download.
  4. Go to MAL Import .
  5. Select MyAnimeList Import.
  6. Upload the XML file.

  4. Upload the XML file.
  5. Click Import.
